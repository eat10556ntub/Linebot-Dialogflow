# A-04 文字外的回覆


```
Intents
   .
   .
   |__ 某個intent
   |       |               
   |       |__ Responses
   |               |
   |               |__ DEFAULT
   |               |     |__ Text response  
   |               |     |__ Custom payload (自訂回應JSON, 以下範例寫在這裡) 
   |               |
   |               |__ LINE  
   |                     |__ 內定的 Text response
   |                     |__ 內定的 Image
   |                     |__ 內定的 Quick replies
   |                     |__ 內定的 Card
   |                     |
   |                     |__ Custom payloads
   .                            |__ (自訂回應JSON, 以下範例寫在這裡)
   .  
```

## 以Line為例:

### (1) 貼圖 (packageId=1, 2, 3或4)

```javascript
{
    "line":{
         "type": "sticker",
         "packageId": "1",
         "stickerId": "1"
    }
}
```



### (2) 圖片

```javascript
{
    "line":{
        "type": "image",
        "originalContentUrl": "自己的圖片位址",
        "previewImageUrl": "自己的圖片位址"
    }
}
```



### (3) 聲音(duration是音檔長度ms)

```javascript
{
    "line":{
        "type": "audio",
        "originalContentUrl": "自己的聲音檔位址",
        "duration": 60000
    }
}
```



### (4) 影片

```javascript
{
    "line":{
        "type": "video",
        "originalContentUrl": "自己的影片位址",
        "previewImageUrl": "自己的圖片位址"
    }
}
```

### (12) 日期選取 (以postback回傳, line對話中不會顯示日期)

```javascript
{
    "line": {
        "type": "template",
        "altText": "這是一個確認樣板",
        "template": {
            "type": "confirm",
            "text": "是否確認要訂房?",
            "actions": [{  
                "type":"datetimepicker",
                "label":"請選擇日期",
                "data":"訂房日期",
                "mode":"date",
                "initial":"2018-11-20",
                "max":"2018-12-31",
                "min":"2018-11-20"
            },
            {
                "type": "message",
                "label": "放棄訂房",
                "text": "否"
            }]
        }
    }
}
```
